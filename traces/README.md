# Trace Files

Place your PCAN .trc trace files here.

The dashboard will automatically detect all .trc files
and make them available in Settings > Trace Replay.

## Example Traces

Copy your traces:
  cp ~/path/to/your/*.trc ~/thinkcity-dashboard-v3/traces/

Supported format:
  PCAN-View .trc files (Version 1.1)
